GPU Cloth

PBD constraints solved on compute; pinned vertices, wind force, simple collision.

← Back to Work

Overview

A grid mesh simulated via Position‑Based Dynamics entirely on the GPU. Distance constraints maintain structural/shear/bend relationships; wind applies external forces, and a simple sphere collider prevents interpenetration.

Highlights

  • Parallel constraint solve via compute work groups
  • Fixed pins and damping for stability
  • Indirect draw with dynamic vertex buffers
